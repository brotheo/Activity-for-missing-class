Step 2 — Collect a new entry

What we're doing: Creating a function that asks the user for a date and note, checks if they're valid, then saves them.

def add_entry():
    raw_date = input("Date (YYYY-MM-DD): ").strip()
    try:
        datetime.strptime(raw_date, "%Y-%m-%d")
    except ValueError:
        print("‚ùå Please use the format YYYY-MM-DD.\n")

def search_entries():
    keyword = input("Keyword to search for: ").strip().lower()

def remove_item():

How it works:

  • Then divide words by that time in minutes to get WPM
